### To install

`make install`

### For watch / hot reloading

`make watch`

### For VSCode

Install `HTMX Attributes` for htmx autocompletion

Install `Tailwind CSS IntelliSense` for tailwindcss completion
AND
Add this to your settings.json
```
"tailwindCSS.includeLanguages": {
    "templ": "html",
},
```
